# NLP TransJakarta (TiJe) App Analysis 🚌🤖

A Natural Language Processing (NLP) project focused on analyzing user feedback, reviews, or public sentiment regarding the TransJakarta (TiJe) mobile application. This project extracts actionable insights to help improve public transportation UI/UX and operational services.

## 📊 Key Features
* **Data Preprocessing:** Custom text cleaning pipeline tailored for Indonesian slang, abbreviations, stop-word removal, and stemming/lemmatization.
* **Sentiment Analysis:** Classification of user reviews into positive, neutral, and negative sentiments.
* **Topic Modeling:** Extraction of core user complaints or praise (e.g., system bugs, bus schedules, UI/UX issues).
* **Insights Dashboard:** Visual representation of key text trends and sentiment distributions.

## 🛠️ Tech Stack
* **Language:** Python
* **NLP & Text Processing:** NLTK, Sastrawi (for Indonesian stemming), SpaCy, or Scikit-learn
* **Data Handling:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
